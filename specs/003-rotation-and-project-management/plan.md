# Implementation Plan: Rotation of Objects & Project Management

**Input**: Feature specification from `specs/003-rotation-and-project-management/spec.md`

## Summary

Extend the Frameer 3D editor with two cohesive capabilities built on top of the existing `SceneElement` state model:

1. **Object Rotation** — Add `rotation_attributes` (Euler x/y/z in radians) to `SceneElement`. Expose a gizmo mode toolbar (Move / Rotate / Scale) in the viewport. Persist rotation through the same `onObjectChange` pipeline used for translation.

2. **Project Management** — Introduce a `Project` entity and `ProjectStore` that namespaces element storage by project ID. Drive the active project from the `?project=<id>` URL query param. Add project creation and navigation to the System panel. All `localStorage` reads use defensive type guards; corrupt or missing data always yields a sane empty default.

## Key Implementation Notes for Task Generation

### P1 — Rotation (User Story 1)

1. Add `rotation_attributes: readonly OriginAttribute[]` to `SceneElement` in `ElementTypes.ts`.
2. Update all factory functions in `PrimitiveFactory.ts` to initialize 3 rotation attributes at 0.
3. Update `syncElement` in `ElementRenderer.ts` to call `mesh.rotation.set(rx, ry, rz)`.
4. Extend `onObjectChange` handler in `ElementPanel.ts` to capture `obj.rotation.{x,y,z}` → `rotation_attributes`.
5. Create `src/scene/GizmoToolbar.ts`; wire into `main.ts`.
6. All existing test files that construct `SceneElement` literals must be updated to include `rotation_attributes: []`.

### P2 — Project Management (User Stories 2 + 3)

7. Create `src/project/` domain: `ProjectTypes.ts`, `ProjectStore.ts`, `ProjectRouter.ts`, `ProjectBootstrap.ts`, `index.ts`.
8. Update `createElementPanel` to accept `projectId: string`; replace `load()`/`save()` with `ProjectStore` calls.
9. Update `createSystemPanel` with new signature accepting `projectRegistry` + callbacks; add Projects folder.
10. Wire `resolveOrCreateProject()` in `main.ts`; pass `projectId` to panels.

### P3 — Project Switching (User Story 4)

11. Project list buttons in `SystemPanel` call `setActiveProjectId(id)` then trigger page reload.
